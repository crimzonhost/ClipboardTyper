# Changelog

All notable changes to ClipboardTyper are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

---

## [1.0.0] — Initial release

- Global hotkey (default Ctrl+Shift+V) to type clipboard into the focused window
- Configurable shortcut, initial delay, and keystroke delay
- Optional “Require double-press to type” and max character limit
- System tray: Type clipboard now, Settings, Exit
- Start at Windows login option
- Settings persisted via SharedPreferences
- MSIX packaging for install/uninstall as a Windows app
- Documentation: user guide, settings, architecture, security, build & release
